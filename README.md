## Course Information

- **Course**: Introduction to Agile Development and Scrum
- **Provider**: IBM DevOps and Software Engineering (Coursera)

- **Status**:  In Progress

## About This Repository

This is my personal sandbox for learning and practicing agile planning concepts. I'm using this space to document my understanding of agile principles, experiment with tools, and build real examples as I progress through the course.

-  Understand Agile and Scrum frameworks
-  Master sprint planning and backlog management

-  Practice with industry-standard agile tools (ZenHub)

## Getting Started with ZenHub

1. **Install ZenHub**: Chrome extension or web app at [zenhub.com](https://zenhub.com)
2. **Connect to Repository**: Link ZenHub to this GitHub repo
3. **Create Boards**: Set up sprint boards and backlogs
4. **Start Planning**: Use the ZenHub board to manage issues and track progress
